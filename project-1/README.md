
# Walmart Store Sales

## Project Summary

This project involved creating a detailed sales dashboard for Walmart stores using Power BI, focusing on year-over-year sales data from February 2010 to October 2012. It provided insights into sales performance during the economic recovery following the 2008 recession, utilizing data modeling, DAX measures, and conditional columns to enhance analysis.

---

## Business Problem

Walmart stores, like other discount retailers, faced fluctuating sales during the post-2008 recession recovery period. Understanding sales trends and factors affecting performance was critical to making informed business decisions and improving store profitability.

---

## Objective

- Analyze year-over-year sales trends across multiple Walmart stores
- Create a dynamic Power BI dashboard for interactive sales insights
- Incorporate economic indicators such as unemployment rates and holiday effects into sales analysis

---

## Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
- Excel
- CSV Data Files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create calendar and refresh date tables using DAX
- Establish relationships between sales and calendar tables
- Develop conditional columns for holiday and unemployment rate flags
- Build and format DAX measures for sales calculations and trends

---

## Key Insights

- Sales trends showed variability influenced by economic recovery post-2008 recession
- Holiday weeks had a measurable impact on store sales performance
- Unemployment rates correlated with fluctuations in weekly sales figures
- Custom sorting of time-based data improved clarity in month-over-month sales analysis
