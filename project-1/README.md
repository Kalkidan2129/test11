## Project Overview

This project involved creating an interactive Power BI dashboard to assess sales performance of 45 Walmart stores in the US between February 2010 and October 2012. It focused on analyzing year-over-year sales trends during the economic recovery following the 2008 recession. The dashboard incorporated data modeling, conditional columns, and custom sorting to provide clear insights into sales dynamics.

---

## Business Problem

Walmart stores, like many discount retailers, experienced fluctuating sales during the post-2008 recession recovery period. Understanding these sales trends was critical to identifying factors impacting store performance and guiding strategic decisions. The challenge was to analyze multi-year sales data effectively to reveal actionable insights.

---

## Objective

- Analyze year-over-year sales trends across multiple Walmart stores.
- Develop an interactive dashboard to visualize sales performance and key factors.
- Incorporate economic indicators such as holiday periods and unemployment rates into the analysis.

---

## Tools & Technologies

- Power BI Desktop
- Power Query Editor
- Excel
- CSV data files
- Power BI Cloud (Colaberry platform)

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop.
- Create relationships between sales data and calendar tables for time-based analysis.
- Add conditional columns to flag holiday weeks and categorize unemployment rates.
- Apply custom sorting to conditional columns and calendar data for accurate visualization.
- Design and deploy an interactive sales dashboard on the Colaberry Power BI Cloud platform.

---

## Key Insights

- Sales trends showed variability influenced by economic recovery phases post-2008 recession.
- Holiday weeks had a measurable impact on sales performance across stores.
- Unemployment rate categories correlated with fluctuations in store sales.
- Custom sorting and data modeling enhanced clarity and usability of sales visualizations.
