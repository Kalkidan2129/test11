
# Walmart Store Sales

## Project Summary

This project involved creating a comprehensive Power BI dashboard to assess sales performance of 45 Walmart stores in the US between February 2010 and October 2012. The analysis focused on understanding sales trends during the recovery period following the 2008 recession, incorporating data modeling, DAX measures, and conditional columns for deeper insights.

---

## Business Problem

Walmart stores, like many discount retailers, experienced fluctuating sales during the economic recovery after the 2008 recession. The challenge was to analyze and visualize these sales trends to identify patterns and factors affecting store performance across different locations and time periods.

---

## Objective

- Import and model Walmart sales data effectively in Power BI
- Create dynamic DAX measures to calculate sales metrics and year-over-year changes
- Develop an interactive dashboard to visualize sales trends and key indicators

---

## Tools & Technologies

- Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)
- Excel
- CSV Data Files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create and refresh calendar and date tables for time intelligence
- Establish relationships between sales and calendar tables
- Develop conditional columns for holiday and unemployment flags
- Build DAX measures for total sales, average sales, and year-over-year growth

---

## Key Insights

- Sales trends varied across stores during the post-recession recovery period
- Holiday weeks and unemployment rates influenced weekly sales performance
- Year-over-year sales growth highlighted stores with stronger recovery
- Custom sorting and calendar management improved temporal analysis accuracy
